![License](https://img.shields.io/badge/License-MIT-green)

![Gameplay Preview](https://github.com/scrnr/Terminal-Coin-Hunter/blob/main/TerminalCoinHunter/assets/gameplay.gif)

A small console game written in C# and .NET 8.

Collect all coins on the map while avoiding an enemy chasing you.
The game contains 10 Normal levels and 10 Hard levels with different maze layouts.

## Features

- Enemy with simple pathfinding AI
- Coin collection gameplay
- Sound effects and mute option
- Console-based rendering

## Controls

| Key | Action |
| :--- | :--- |
| `↑ / ↓ / ← / →` | Move |
| `Enter` | Select / Next Level |
| `R` | Restart Level |
| `M` | Mute / Unmute Sound |
| `Esc` | Exit |

3. Extract the archive.
4. Run `TerminalCoinHunter.exe`
